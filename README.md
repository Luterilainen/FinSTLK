FinSTLK raamattumoduuli OSIS XML lähdemateriaali
------------------------------------------------

OSIS-tiedoston syntaksin voi tarkistaa komennolla

```
xmllint --noout --schema osisCore.2.1.1.xsd FinSTLK.xml
```

Sword-moduulin (zip-tiedosto) voi kääntää komennolla

```
bash build_sword.sh
```
